## 🚀 Kom i Gang

1. **Klone prosjektet**
```bash
git clone https://github.com/Marcus-Kodehode/Match-A-Gift.git
cd Match-A-Gift
2. **Installer avhengigheter**
```bash
npm install
```

3. **Start utviklingsserver**
```bash
npm start
```

4. **Åpne nettleseren**
```
http://localhost:4200
```
